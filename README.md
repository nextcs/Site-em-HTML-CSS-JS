🌐 DevLinks - Página de Links Pessoais

Este projeto é uma página web pessoal desenvolvida para centralizar e compartilhar links importantes, como portfólio e redes sociais, de forma simples, moderna e responsiva.

🚀 Tecnologias utilizadas
HTML5
CSS3
JavaScript
Google Fonts (Inter)
Ionicons

📸 Preview
Uma página com:

Foto de perfil
Nome de usuário
Links para GitHub e LinkedIn
Ícones sociais
Alternância entre modo claro e escuro

🎯 Funcionalidades

Layout responsivo (funciona em mobile e desktop)
🌙 Modo escuro (dark mode)
☀️ Modo claro (light mode)
Troca dinâmica de imagem de perfil conforme o tema
Estilização moderna com uso de variáveis CSS
Efeitos visuais e animações suaves

🧠 Como funciona o Dark/Light Mode
O tema é alterado através de JavaScript, que adiciona ou remove a classe light no elemento <html>.
Com isso, variáveis CSS são atualizadas automaticamente, mudando:

Cores
Background
Ícones
Imagem de perfil
